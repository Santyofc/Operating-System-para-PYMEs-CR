# Project Brief

Objetivo: construir un Operating System SaaS para PYMEs en Costa Rica.

Componentes:
- ERP (ventas, inventario)
- Facturación electrónica
- CRM
- Automatización

Requisitos:
- Multi-tenant
- Modular
- Escalable

El sistema debe permitir añadir módulos independientes conectados por un hub central.
